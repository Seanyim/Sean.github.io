# Sean's Space - 个人主页重构

根据您的要求，我已经使用现代科技风 + 个人特色的设计语言重新设计了您的个人主页。

## 🌟 新特性

1.  **全新设计系统**：
    - **配色**：极简黑白灰 (`#18181B`) 搭配 科技蓝 (`#2563EB`) 点缀。
    - **字体**：标题使用手写体 `Caveat` 展现个人风格，正文使用 `Quicksand` 保持现代易读性。
    - **动效**：卡片悬浮、按钮微交互、平滑过渡动画。

2.  **功能模块**：
    - **首页 (`index.html`)**：展示个人简介、精选项目（Projects）、关于我（About）。
    - **博客 (`blog.html`)**：长文写作平台。
      - 支持富文本编辑（加粗、列表、链接、图片）。
      - 文章直接保存在本地浏览器中（LocalStorage）。
    - **微博 (`microblog.html`)**：碎碎念/动态发布平台。
      - 支持快速发布文字、图片链接、表情。
      - 仿社交网络的时间流布局。

3.  **主要文件结构**：

    ```
    Sean.github.io/
    ├── index.html        # 首页
    ├── blog.html         # 博客页
    ├── microblog.html    # 微博页
    ├── css/
    │   └── style.css     # 全局样式表（设计系统核心）
    ├── js/
    │   └── main.js       # 交互逻辑 & 数据存储
    └── design-system/    # 设计系统文档（可参考 MASTER.md）
    ```

## 🚀 如何使用

1.  **直接打开**：双击 `index.html` 即可在浏览器中预览。
2.  **写博客**：
    - 进入 Blog 页面，点击 "New Article"。
    - 在弹出的编辑器中写作并发布。
3.  **发微博**：
    - 进入 WeiBo 页面。
    - 在顶部输入框输入内容，点击下方的图标添加图片链接或表情。
    - 点击 "Post" 发布。

## ⚠️ 注意事项

- **数据存储**：目前博客和微博的数据保存在您浏览器的 **LocalStorage** 中。这意味着：
  - 数据仅在当前浏览器可见。
  - 清除浏览器缓存会丢失数据。
  - （这是为了演示静态页面的动态交互能力，无需后端服务器）。
- **图片**：由于是纯前端演示，上传图片功能是通过“粘贴图片URL”实现的。您可以直接复制网络图片的地址填入。
