RISC-V Efficient Core

High-Performance Edge Computing Architecture

RV64IMAC

Tech Stack

Verilog/SystemVerilog C++ FPGA (Xilinx) Chisel

Overview

This project involves the design and implementation of a 5-stage pipelined RISC-V processor core. The goal was to optimize for performance-per-watt in edge AI scenarios.

Technical Details

  • ✅ ISA Support: RV64IMAC (Integer, Multiply, Atomic, Compressed).
  • ✅ Pipeline: 5-Stage (Fetch, Decode, Execute, Memory, Writeback) with forwarding unit.
  • ✅ Branch Prediction: Dynamic 2-bit saturating counter + BTB.
  • ✅ Memory System: Direct-mapped I-Cache and 2-way set-associative D-Cache.

Outcome

Successfully synthesized on Xilinx Artix-7 FPGA running at 100MHz. Capable of booting custom RTOS and running quantized neural network inference.
